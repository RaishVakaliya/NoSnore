## Features

- **Prevent Cold Starts**: Keep your free-tier backends alive with automated scheduled pings.
- **Service Management**: Create, update, and monitor up to 10 service endpoints (Pro).
- **Custom Intervals**: Flexible scheduling—15m for Free, down to 1m or 5m for Pro users.
- **Secure Authentication**: Robust user management and profile syncing powered by Clerk.
- **Pro Upgrades**: Instant tier upgrades via Razorpay for expanded limits and faster pings.

## Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Database & Backend:** [Convex](https://www.convex.dev/) (Real-time sync & serverless functions)
- **Authentication:** [Clerk](https://clerk.com/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/)
- **Payments:** [Razorpay](https://razorpay.com/)
- **Deployment:** [Vercel](https://vercel.com/)
